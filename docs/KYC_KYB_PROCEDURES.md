# Procedimentos KYC/KYB

Este documento define procedimentos minimos para verificacao de usuarios e empresas.

## KYC (PF)
- Niveis: UNVERIFIED, BASIC, FULL.
- Dados minimos (BASIC): nome, email, documento, data nascimento.
- Dados completos (FULL): endereco, selfie/liveness, comprovante.
- Revalidacao periodica para FULL.

## KYB (PJ)
- CNPJ/registro, contrato social, beneficiario final (UBO).
- Identificacao de administradores e signatarios.
- Comprovantes de endereco e atividade.
- Evidencias guardadas para auditoria.

## Controles obrigatorios
- Screening de sancoes e PEPs no onboarding e periodicamente.
- Bloqueio ou escalonamento em caso de match.
- Registro de decisao, justificativa e evidencias.

## Saidas
- Resultado: aprovado, reprovado, pendente.
- Perfil regulatorio atualizado e auditavel.
